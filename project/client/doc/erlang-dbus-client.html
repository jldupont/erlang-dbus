Description

Messages Generated

The following messages are generated back to the Client.

An error occured:

     {edbus, {error, Reason}}

The Client interface is ready to be used:

     {edbus, {ready, UName}}

The Client attempted to use the interface prematurely:

     {edbus, {error, 'interface.not.ready'}}
